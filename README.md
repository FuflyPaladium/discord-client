# Discord Desktop Client Sources

---

**Install :**
```console
$ git clone https://github.com/FuflyPaladium/discord-client/
$ cd discord-client
$ npm install && npm i electron --save-dev
```

---

**Start :** 
```console
$ npm start
```

---

**Build :**
```console
$ npm run dist
```
